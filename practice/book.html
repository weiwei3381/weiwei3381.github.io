# 高效Node.js——基础指南

📘《Efficient Node.js: A Beyond-the-Basics Guide》，2025年6月发布

ISBN: 978-1098145194 | 240页 | PDF, EPUB | 20 MB

使用 Node.js，将您的 Web 开发技能从浏览器迁移到服务器。Node.js 是一个流行的后端框架，亚马逊、Netflix、LinkedIn 等公司超过 1000 万名开发者都在使用它。如果您熟悉 JavaScript，Samer Buna 的这本实用指南将向您展示如何有效地构建和维护最复杂的 Node.js 应用程序。

通过基于实践的、基于项目的方法，您将从关键基础知识转向高级概念，例如模块、包、事件驱动架构、流、子流程、扩展、测试、部署等等，同时专注于实践中真正重要的事情。

网盘下载：efficient-nodejs-beyond-basics.rar (opens new window)

# React之路（2025年版）

📘《The Road to React: The React.js in JavaScript Book (2025 Edition)》，2025年6月8日发布

ISBN: 978-1720043997 | 285页 | PDF, EPUB | 10 MB

《React之路：在 JavaScript 中掌握 React.js 的旅程》是一本全面、实用且简洁的 React feat.Hooks 书籍。

在《React 之路》中，你将学习 React.js 的所有基础知识以及 Hooks，并逐步构建一个功能齐全的 React 应用程序。在创建 React 应用程序的过程中，每一章都会向你介绍一个新的 React 关键功能。然而，本书不仅涵盖基础知识：本书还深入探讨了相关主题（例如，使用 TypeScript 编写 React、测试、性能优化）以及客户端和服务器端搜索等高级功能的实现。读完本书，你将拥有一个已部署且功能齐全的 React 应用程序。

网盘下载：road-react-robin-wieruch.rar (opens new window)

# 学习Vue——可重用、可组合、可扩展用户界面的核心概念和实用模式

📘《Learning Vue: Core Concepts and Practical Patterns for Reusable, Composable, and Scalable User Interfaces》，2024年1月发布

ISBN: 978-1492098829 | 352页 | PDF, EPUB | 45 MB

学习 Vue.js 的核心概念，这是一个用于从零开始构建前端应用程序和界面的现代 JavaScript 框架。本书通过简洁、实用且清晰的示例，逐步指导 Web 开发人员了解 Vue.js 生态系统中的工具和库，以及如何使用它们为实际的 Web 项目创建完整的应用程序。

作者 Maya Shavin 讲解了如何使用 Vue.js 库中的各种工具创建完整的应用程序开发流程。您将学习如何使用 Pinia 架构处理组件之间的数据通信，为前端项目开发可管理的路由系统以控制应用程序流程，以及如何制作基本的动画效果以打造更佳的用户体验。

网盘下载：learning-vue-core-concepts-practical.rar (opens new window)
