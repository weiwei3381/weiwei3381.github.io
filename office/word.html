# Word和Pdf使用技巧

# Word快捷键技巧

主要记录一些不太常用但是还蛮有意思的操作和快捷键。

# F8的功能

按F8进入“扩展选定区域模式”，在扩展选定区域模式下，单击文档中的位置将把当前选定内容扩展到该位置，这样就不用担心选取内容太长前面跑掉了。重复按F8将所选内容扩展到整个字词、句子、段落、章节和文档，按Shift+F8减少选定内容，按Esc停止扩展所选内容。

# 文本对齐

  • 段落居中：Ctrl+E
  • 两端对齐：Ctrl+J
  • 左对齐：Ctrl+L
  • 右对齐：Ctrl+R

# 应用样式

  • 应用“普通（Normal）”样式：Ctrl+Shift+N
  • 应用“标题 1”样式：Ctrl+Alt+1
  • 应用“标题 2”样式：Ctrl+Alt+2
  • 应用“标题 3”样式：Ctrl+Alt+3

# 调整字体

  • 显示字体对话框: Ctrl+Shift+F
  • 增大字号：Ctrl+Shift+右尖括号 (>)
  • 减少字号：Ctrl+Shift+左尖括号 (<)
  • 在大写、小写和首字母大写之间切换：Shift+F3
  • 显示“展现格式”任务窗格：Shift+F1

# 插入特殊字符

  • 插入换行符：Shift+Enter，“换行符”和“回车”的区别在于：“换行符”的逻辑上还是同一个段落，不过视觉上换到下一行去了；“回车”的话则直接插入新段落。
  • 插入分页符：Ctrl+Enter
  • 插入样式分隔符：Ctrl+Alt+Enter,样式分隔符的作用是使一个段落内实际包含两个段落，实现两个段落样式

# 更改视图

  • 切换大纲（Outline）视图: Ctrl+Alt+O
  • 切换页面（Page）视图: Ctrl+Alt+P
  • 切换草稿视图: Ctrl+Alt+N
  • 切换到打印预览：Ctrl+Alt+I

# 使用域

  • 插入空域：Ctrl+F9
  • 插入页字段（Page）：Alt+Shift+P，用这个可以直接插入页码，word的“插入页码”功能本质上就是用这个域实现的。
  • 插入“日期”域（Date）：Alt+Shift+D
  • 插入“时间”域（Time）：Alt+Shift+T
  • 插入空域: Ctrl+F9
  • 更新选定的域: F9，因此想要更新全文的域，先全选再按F9
  • 在所选的域代码及其结果之间进行切换：Shift+F9
  • 在所有的域代码及其结果间进行切换：Alt+F9
  • 前往下一个域：F11
  • 定位至前一个域：Shift+F11

# 其他

  • 重复上一个命令或操作：F4
  • 显示“定位”对话框：F5
  • 将光标移动到上次更改：Shift+F5
  • 检查文档或所选文本中的拼写和语法：F7
  • 显示书签对话框：Ctrl+Shift+F5
  • 显示另存为对话框：F12
  • 显示转到对话框：Ctrl+G
  • 显示“样式”任务窗格：Ctrl+Alt+Shift+S
  • 显示字体对话框：Ctrl+D 或 Ctrl+Shift+F
  • 显示段落对话框：Alt+O+P

# 问题解决方案

# word2010不能插入页码

问题描述: word2010插入页码, 发现并没有页码模板
解决方案:

  1. 打开word, 点击文件按钮
  2. 点击选项按钮
  3. 点击加载项
  4. 点击管理下拉框,选择模板, 点击转到
  5. 在模板和加载项面板, 点击添加
  6. 在浏览对话框中, 找到Built-In Building blocks.dotx文件, 这个文件一般放在C:\Program Files\Microsoft Office\Office14\Document Parts\2052\14中, 然后点击确定就可解决问题.

# 扫描PDF插入页码

  1. 使用acrobat编辑文件, 如果没有显示"内容编辑""页面"等工具, 则点击右侧工具按钮
  2. 点击页面, 选择页眉和页脚
  3. 根据需要选择页脚的"左中右"三个输入框, 点击插入页码, 字体和页边距可以调整页码大小和位置, 确定后就可以看到插入的页码。

# 删除尾注横线

  1. 切换草稿视图：点击视图->草稿；
  2. 打开备注窗口：点击引用->显示备注；
  3. 在备注窗口删除横线：点击底部窗口中的下拉菜单，选择脚注分隔符项，鼠标选中横线，按键盘Delete键删除。

# Word中的eq域详解

文中技巧大部分来源于bilibili的Up主一闪流溢 (opens new window)。

Word域分为域代码和域值两个部分，例如插入的页码就是域代码，选中页码之后使用“shift+F9”可以切换显示样式，页码的域代码就是{page \* MERGEFORMAT}。

域代码中, page指的是该域是page域, 然后\后面跟的就是这个域对应的参数,参数不区分大小写, 以页码为例, 如果我们想让他按照3位数字显示, 即"1"显示为"001", 则使用的域代码就是{page \#000}

页码是page域,目录是toc域,点击word的"插入-文档部件-域"可以看到word的所有域,如下图所示.

如果自己写域代码,则先使用快捷键Ctrl+F9进入域编辑模式, 此时会弹出一个大括号, 然后在大括号中间书写域代码, 例如word文件所在位置的域代码则是{filename \p}, 选中之后点击右键"切换域代码"也可显示域值计算结果.

# EQ域的分数显示

分数(Fraction, fenshu)的开关参数为 \F, 基本用法是{eq \f(分子,分母)}典型示例为:

  • 12\frac{1}{2}​2​​1​​的域代码为{eq \f(1,2)}
  • 24=12\frac{2}{4}=\frac{1}{2}​4​​2​​=​2​​1​​的域代码为{eq \f(2,4) = \f(1,2) }, 其中=的左右空格不能少. 也可以用两个eq域来进行显示,中间的等号为普通字符, 即: {eq \f(2,4)}={eq \f(1,2)}
  • 分子和分母也不仅限于数字,例如: \frac{我的}{心情不错}的域代码为{eq \f(我的,心情不错)}

# EQ域的数组显示

数组(Array)的开关参数为\A,数组内部的典型参数还有以下这些:

数组的默认排列为按行排列,纵向显示,典型示例为:

  • 12\begin{matrix} 1\\2 \end{matrix}​1​2​​ 表示为{eq \a(1,2)}
  • 123\begin{matrix} 1&2&3 \end{matrix}​1​​​2​​​3​​因为要排成3列,所以用排n列的参数开关\con,其中n是要排的列数, 表示为{eq \a\co3(1,2,3)},然后需要把列的间隔打开,那么再加上列间距的参数开关\hsn,n越大则间距越大, 表示为{eq \a\co3\hs8(1,2,3)}
  • 123456789\begin{matrix} 1&2&3\\4&5&6\\7&8&9 \end{matrix}​1​4​7​​​2​5​8​​​3​6​9​​的显示为{eq \a\co3\hs10\vs10(1,2,3,4,5,6,7,8,9)}

# EQ域的列表显示

列表(List)的开关为\L, 与数组不同, 列表是横向显示,项目与项目之间用逗号分隔, 因为这个特性, 所有要显示为逗号分隔的项目都可以在前面加上\l来将括号去掉, 因此通常与其他符号配合使用,例如:

  • a,b,ca,b,ca,b,c表示为{eq \l(a,b,c)}
  • a,bc,de,fg,h\begin{matrix} a,b&c,d\\e,f&g,h \end{matrix}​a,b​e,f​​​c,d​g,h​​表示为{eq \a\co2\hs5\vs5(\l(a,b),\l(c,d),\l(e,f),\l(g,h))}

# seq域（序列域）

seq域常常用于自动增长的序列表示，比如论文中的图号、表号，{seq name}默认参数为序列名称。命令参数包括：
\n，“number”表示序列号较之于同一个域的序列号递增，默认自带；
\c，"const"表示序列号不变；
\h，“hide”隐藏域结果；这可以用于声明一个域，比如在论文中，在章节开始出插入一个 \h代码，章节开始处不会有序列号，但是后面的图片标题可以依据这一个章节域进行递增
\r，“reset”重新设置序号，例如{seq list \r 1}表示重新设置从1开始。
\s，“section”在指定标题级别下面重新开始，例如seq list \s 1表示从一级标题重新开始，主要用于多级标题使用。
\*，指定标号格式，可选格式很多，可以选中域->鼠标右键->编辑域->选项，选择需要的格式，自动生成相应的域代码，典型的比如chinesenum1、Arabic、roman等

  • {seq list}：表示定义了一个名为list的序列，默认是阿拉伯数字1、2、3、4等以此类推。

中文编号使用{seq a\* chinesenum3}

# Word通配符查找替换

文中技巧来自B站Word通配符教程 (opens new window)

在正文中输入=rand()然后回车能够产生随机段落，可以帮助进行测试。

高级查找替换进入方式按ctrl+H，然后点查找，在查找时可以选择在以下项中查找-主文档，可以一次查找所有的匹配内容；有些时候需要多次查找定位时，可以先按照主文档查找，然后在在以下项中查找-当前所选内容再用其他的匹配方法查一遍。

典型事例1：需要定位以第开头的段落，并且将整个段落全部选中，那么就可以先使用^13第*^13在主文档中查找，选中以段落结束开头后面接“第”，然后的整段。接下来，使用第*^13选择在“当前所选内容”中进行查找，这样就可以把之前的换行符排除掉了。

典型事例2：避免搜索时跨段落，当查找通配符中带*时，很有可能出现跨段落搜索情况，这时为了避免这种情况发生，可以先用*^13在“主文档”中查找所有段落，然后在“当前所选内容”中查找对应的字符，就不会出现跨段落的情况了。

# Word中内容通配

1.回车：硬回车使用^p（在开启“使用通配符”选项后使用^13），软回车使用^l（在开启“使用通配符选项后使用^11）；典型的应用包括把多个空行统一变为1个。

2.空格：直接使用空格即可匹配；

3.制表符：查找符号为^t；

4.数字：多个数字中匹配任意一个使用中括号来匹配，即：[13579]，匹配所有的数字可以使用[0-9]来简便表示。

5.任意字符：使用星号*来表示，包括没有字符的情况；

6.任意单个字符：使用问号?表示，确定的多个单个字符可以在后面加{}，例如任意2个或者3个字符则使用?{2,3}；

7.任意段落：使用*^13勾选使用通配符，比如要给每个段落后面加一个空行，那么可以用*^13替换为^&^13；

8.英文单词和字母：使用[a-zA-Z]匹配所有的英文字母，用[a-zA-Z]{1,}匹配所有超过1个字母的单词，同时还能使用<>分别匹配单词的开始和结尾，可以单独使用，例如<b?d>能够匹配“bed”但是不会匹配“bedroom”，理论上来说使用<b[a-zA-Z]{1,}d>可以匹配单词，但是这里有个问题，如果要查找的内容在范围里面，就不能用{}，例如这里d就在[a-zA-Z]里面，而是要改为@，表示前1个或者多个。

9.中文汉字：可以使用[一-龥]（4E00-9FA5）和[㐀-﨩]（3400-FA29），输入龥的话可以先输入9FA5然后按Alt+X即可，这个方法可以显示对应unicode编码的汉字。其中[一-龥]指的是“CJK统一汉字区”，而[㐀-﨩]的范围更广，表示“CJK兼容汉字区+统一汉字区+专用汉字区”，一般使用CJK统一汉字区就够了，详细表示如下图所示：

例如在文档中搜索所有包含“科技”的一句话，使用<[㐀-﨩]@科技[㐀-﨩]@>进行搜索，这里需要注意的是，由于关键词科技包括在[㐀-﨩]中，所以必须使用@而不是{1,}，而如果不加<>那么到科技的前一个字符和后一个字符就停止了，到不了句前和句后。

9.任意图片：使用^g进行表示。

# Word通配符中涉及的特殊符号

1.[-]：查找范围内的字符，例如同时查找软回车和硬回车就用[^11^13]即可；

字符是否连续的判断方法

比如表示数字[0-9]能够用-表示范围的原因是因为在unicode字符集中，9在0的后面，然后中间正好都是数字，查看unicode编码的方法是在插入-符号-其他符号，弹出的“符号”对话框中，能看到所有的字符、unicode字符代码和位置，其中0123456789都是连着的，所以能用这个方式，同理[A-Z]也是可以的，因为在字符中他们也是连着的。

2.{,}：前一个字符的出现次数，例如[0-9]{1,3}就表示匹配所有1到3位的数字，其中3也可以不写，即[0-9]{1,}，表示匹配1位以上的数字；

3.^&：查找到的内容，即之前查找到的东西，比如可以批量给词语“语文”改为“《语文》”，替换方法为：语文，替换为：《^&》，也可以用于给指定内容替换为对应的格式。

4.!：表示非，主要用在[]中，比如[!0-9]表示查找所有的非数字。

5.\：转义符使用\表示，例如匹配[]需要使用\[\]

6.^c：表示系统剪贴板内容，像域、分节符、图片等内容无法再替换栏输入，就可以先复制，之后用^c进行替换。

# word表达式查找替换

表达式指的是一组英文括号()，即查找的时候可以指定里面的一部分内容，而且可以指定多组，在替换的时候如果是第一组就是\1，第二组就是\2，以此类推。

例如：将所有的书名号内容替换为用双引号表示，例如《西游记》《三国演义》替换为“西游记”“三国演义”，查找的内容是：《(*)》替换为“\1”。
